# Poimenexams

A modern online examination platform designed for educational institutions, churches, training organizations, and professional development programs.

ExamForge enables teachers and administrators to:

- Create exams
- Manage question banks
- Assign assessments to groups
- Track performance
- Apply negative marking
- View analytics
- Generate AI-powered questions from books (future feature)

---

## Features

### MVP

- User Authentication
- Question Bank
- Exam Builder
- Timed Exams
- Automatic Scoring
- Negative Marking
- Student Results
- Teacher Dashboard

### Future Features

- PDF Upload
- EPUB Upload
- AI Question Generation
- Certificates
- Email Notifications
- Advanced Analytics
